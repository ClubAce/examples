#Примеры webOS smartTV
Данный репозиторий содержит набор примеров web приложений с использованием технологий:
* HTML5
* Luna сервисы
* Cordova фреймворк 

##HTML5
* **html5HelloWorldHosted** - Пример *hosted* приложения 
* **html5HelloWorldPackaged** - Пример *packaged* приложения
* **html5BackButton** - Как отловить клавишу *back*
* **html5InputText** - Демонстрация виртуальной клавиатуры и получение кодов спец клавиш (*enter*, *back*)
* **html5videoPlayer** - Пример работы стандартного html5 видео плеера

##Luna сервисы
* **lunaTVDeviceInformation** - Получение информации о ТВ на примере `com.webos.service.tv.systemproperty`
* **lunaApplicationManager** - Запуск приложений по `id`
* **lunaLaunchRelaunchEvents** - Обработка событий `webOSLaunch` и `webOSRelaunch`
* **lunaDRM** - Пример воспроизведения playready и widevine DRM
* **lunaNodeJsService** - Пример создания пользовательских nodejs сервисов
* **lunaNodeExtension** - Расширение функционала nodejs сервисов на примере модуля md5

Последняя версия библиотеки webos.js [доступна на GitHub](https://github.com/webOS-DevRel/webOS.js)

##Cordova
* **cordovaAccelerometer** - Получение данных сенсора акселерометра с MRCU

Библиотека Cordova [доступна здесь](http://developer.lge.com/webOSTV/index.php/download_file/view/912/2381/)
